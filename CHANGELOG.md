# Changelog

All notable changes to PeekDock are documented here.

## 0.1.0 - 2026-07-21

- Initial AutoHotkey v2 release.
- Bind the active Chrome tab with `Ctrl + Alt + Shift + B`.
- Toggle the dedicated app window with the middle mouse button.
- Toggle always-on-top with `Ctrl + Alt + T`.
- Keep the restored window size and position unchanged.
- Store local runtime state in `config.ini`.
